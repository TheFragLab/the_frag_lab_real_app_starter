
# The Frag Lab Alpha

A real React/Vite starter app for The Frag Lab.

## What it includes
- The Bench
- MIX button
- Lab Reports / live experiment tracking
- The Canon search
- I Wonder board
- Local storage persistence
- JSON backup export

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Upload to Netlify, Vercel, Cloudflare Pages, or GitHub Pages.
